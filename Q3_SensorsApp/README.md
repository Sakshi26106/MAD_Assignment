Sensor Data App (Q3)

This Android application reads and displays real-time data from device sensors:

* Accelerometer (X, Y, Z axis values)
* Light Sensor (ambient brightness)
* Proximity Sensor (distance from object)

Features:

* Real-time sensor updates
* Simple and clean UI

Technologies Used:

* Java
* Android SensorManager API

Note:
Sensor availability may vary depending on the device hardware.
